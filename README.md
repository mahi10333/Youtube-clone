# Youtube-clone

A fully responsive YouTube homepage clone built using HTML5 and CSS3. This project recreates the core user interface and layout of YouTube while focusing on responsive web design, modern UI structuring, and frontend development fundamentals.

Features
Responsive YouTube-inspired layout
Navigation sidebar
Search bar interface
Video thumbnail grid system
Channel profile sections
Hover effects and interactive styling
Clean and structured UI design
Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
Responsive Web Design

Project Objective
The main objective of this project was to strengthen frontend development skills by building a real-world UI clone and improving understanding of layout structuring, responsiveness, spacing, alignment, and modern CSS techniques.

Learning Outcomes
Through this project, I improved my knowledge of:
Responsive layouts
Flexbox and CSS Grid
UI structuring
Component-based page design
Modern frontend styling practices
Future Improvements
Add JavaScript functionality
Implement dark mode
Create responsive mobile menu
Add video playback page
Integrate YouTube API
